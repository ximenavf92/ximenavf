When designing for web you have to be aware of how that design wil behave in the browser. [CodePen](https://codepen.io) has been a very useful tool to take a design idea and prototype it to asses the functional and visual possibilities before implementing it at scale. It has also been a great way to test and debug pieces of code without affecting the main project. Here are a few examples.
